0x00-shell_basics
This directory contains the Shell Basics project for the System Engineering and DevOps course.

The following scripts are included:

0-current_working_directory – Prints the absolute path of the current working directory.
1-listit – Lists the contents of the current directory.
2-bring_me_home – Changes the current directory to the user’s home directory.
3-listfiles – Lists the contents of the current directory in long format.
4-listmorefiles – Lists all contents, including hidden files, in long format.

Each script starts with #!/bin/bash and is executable.
